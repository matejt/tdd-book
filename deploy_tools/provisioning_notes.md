Provisioning a new site
=======================

## Required packages
* nginx
* Python 3.6
* virtualenv + pip
* git


## Folder structure
Assume we have a user account at /home/username

/home/username
|_ sites
    |_domain1
        |_.env
        |_db.sqlite3
        |_static
        |_ etc...
    |_domain2
        |_.env
        |_db.sqlite3
        |_static
        |_ etc...
